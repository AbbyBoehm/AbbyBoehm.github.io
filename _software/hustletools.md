<div style="text-align: justify; text-justify: inter-word;">
  <p><a href="https://github.com/Exo-TiC/HUSTLE-tools">HUSTLE-tools</a> is a one-stop shop for 1D spectral extraction from G280 time-series observations. Developed by the HUSTLE team to support community use of the HST WFC3/UVIS G280, our open-source and user-friendly pipeline is designed to be easily run with minimal prior code experience. HUSTLE-tools also uses a configuration file system with extension .hustle which makes it easy for users to repeat and share their data reduction processes, enabling efficient exploration of reduction options for your G280 data.</p>
  
  <p>Check out the documentation for HUSTLE-tools on <a href="https://hustle-tools.readthedocs.io/en/main/">Read the Docs</a>!</p>
